# @tsd-ui/table-controls

Table control contracts and hooks composing `@tsd-ui/core` logic with PatternFly table components.

## Installation

```bash
npm install @tsd-ui/table-controls
```

This will also install `@tsd-ui/core` as a dependency.

### Peer dependencies

```bash
npm install react
```

| Peer | Version |
|------|---------|
| `react` | `^18.0.0` |

## Exports

- `ITableControlsConfig` — configuration interface for table controls
- `ITableControlDerivedState` — derived state from table control hooks
