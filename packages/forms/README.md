# @tsd-ui/forms

Form-oriented wrappers composing `@tsd-ui/core` logic with PatternFly form components.

## Installation

```bash
npm install @tsd-ui/forms
```

This will also install `@tsd-ui/core` as a dependency.

### Peer dependencies

```bash
npm install react
```

| Peer | Version |
|------|---------|
| `react` | `^18.0.0` |

## Exports

- `IFormGroupWrapperProps` — props interface for form group wrappers
- `IFormPersistenceOptions` — configuration for form state persistence
